# Madrid Website

Welcome to the Madrid Website project! This project is a simple HTML website that displays a welcoming message for visitors.

## Project Structure

```
madrid-website
├── src
│   └── index.html
├── .github
│   └── workflows
│       └── deploy.yml
└── README.md
```

## Files

- **src/index.html**: Contains the HTML structure of the website with the message "Welcome to Madrid".
- **.github/workflows/deploy.yml**: Defines the GitHub Actions workflow for deploying the website.

## Setup Instructions

1. Clone the repository to your local machine.
2. Navigate to the `madrid-website` directory.
3. Open `src/index.html` in your web browser to view the website.

## Deployment

The website is automatically deployed using GitHub Actions whenever changes are pushed to the repository. Ensure that the workflow file is correctly configured in `.github/workflows/deploy.yml`.
